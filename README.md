# hostedsuite-cli

Command-line interface for **HostedSuite** — built for support staff and AI agents (Claude Code).
Installs the `hs` binary and a paired Claude Code skill.

> 🚧 **Under active development — not yet released.**

## What it does

`hs` is a thin, fully self-documenting CLI over the HostedSuite REST API: look up and correct client,
contact, center and reservation data, run reports, and perform guarded bulk updates across tenant
deployments. Every command is documented via `--help`, and a companion Claude Code skill teaches
agents how to drive it safely — with confirmation gates on bulk operations so the wrong tenant
doesn't get changed by mistake.
